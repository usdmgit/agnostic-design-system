# Dropdown Component

## Type: Composite

The Dropdown contains the following components: [Button](https://design-system.codelitt.dev/?path=/docs/components-button--neutral), [Input](https://design-system.codelitt.dev/?path=/docs/components-input--default) and [List](https://design-system.codelitt.dev/?path=/docs/components-list--simple-list).

**Category**

1. [Simple](https://design-system.codelitt.dev/?path=/story/components-dropdown--default-dropdown) - default appearance of the Dropdown.
2. [Icon](https://design-system.codelitt.dev/?path=/story/components-dropdown--icon-dropdown) - displays an image icon in the Dropdown field.

**List positions**

1. bottom
2. top

  You can use the CSS variables ```--dropdown-list-top``` to control the position of the list in case you selected the 'bottom' option<br />
or ```--dropdown-list-bottom``` in case you selected the 'top' option

**Positions**

1. left
2. right

## Getting Started

```javascript
import React, { useState } from 'react';
import { Dropdown } from '@codelittinc/agnostic-design-system';

const SimpleDropdown = () => {
  const [selected, setSelected] = useState();
  const options = [
    {
      label: 'Option 1',
      value: '1'
    },
    {
      label: 'Option 2',
      value: '2'
    }
  ];

  return (
    <Dropdown
      id='ads-dropdown'
      category='simple'
      selectorText='Select an option'
      selected={selected}
      options={options}
      onChange={setSelected}
      getItemLabel={item => item.label}
      getItemKey={item => item.value}
      getItemValue={item => item.value}
      getListTitle={item => item.label}
    />
  );
};

export default SimpleDropdown;
```

## CSS Variables available

Given the nature of a composite component, in addition to its own, the Dropdown utilizes CSS variables from the Button, Input and List components.
They are listed in the table below.

The CSS variables ```--dropdown-list-top``` and ```--dropdown-list-bottom``` are used to control the dropdown list positioning.<br />
Use the CSS variable ```--dropdown-list-top``` to control the list positioning if listPosition was defined as 'bottom' or if no option for it was selected.<br />
Use the CSS variable ```--dropdown-list-bottom``` to control the list positioning if listPosition was defined as 'top'.<br />
