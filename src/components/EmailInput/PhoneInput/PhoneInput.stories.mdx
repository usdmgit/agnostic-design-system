## Getting Started

```javascript
import React, { useState } from 'react';
import { PhoneInput } from '@codelittinc/agnostic-design-system';

const SimplePhoneInput = () => {
  const [value, setValue] = useState('');

  return <PhoneInput onChange={phone => setValue(phone)} label='Phone Input' value={value} />;
};

export default SimplePhoneInput;
```

## CSS Variables Available

Although it is not a decorator component, we use all the CSS variables of the [Input](https://design-system.codelitt.dev/?path=/docs/components-input--default) component. Please refer to the previous for more details.
If you need to specify or customize the input with your own class, the component will receive an variablesClassName, which will allow you to overwrite the styles of our default input compontent.
