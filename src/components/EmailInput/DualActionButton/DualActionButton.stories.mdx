# DualActionButton Component

## Type: Simple

## DualActionButton Props

|        Prop         |    Type    | Default | Mandatory |
| :-----------------: | :--------: | :-----: | :-------: |
|  backButtonAction   | () => void |    -    |   true    |
| forwardButtonAction | () => void |    -    |   true    |
|        text         |   string   |    -    |   true    |
| variablesClassName  |   string   |    -    |   false   |
|  disableLeftButton  |  boolean   |    -    |   false   |
| disableRightButton  |  boolean   |    -    |   false   |

## Getting Started

```javascript
import React from 'react';
import { DualActionButton } from '@codelittinc/agnostic-design-system';

const MyComponent = () => {
  return (
    <DualActionButton
      text={'DualActionButton'}
      backButtonAction={() => {}}
      forwardButtonAction={() => {}}
    />
  );
};

export default MyComponent;
```
