# Dropdown Component

## Type: Composite

The Dropdown contains the following components: [Button](https://design-system.codelitt.dev/?path=/docs/components-button--neutral), [Input](https://design-system.codelitt.dev/?path=/docs/components-input--default) and [List](https://design-system.codelitt.dev/?path=/docs/components-list--simple-list).

## Getting Started

```javascript
import React, { useState } from 'react';
import { Dropdown } from '@codelittinc/agnostic-design-system';

const SimpleDropdown = () => {
  const [selected, setSelected] = useState();
  const options = [
    {
      label: 'Option 1',
      value: '1'
    },
    {
      label: 'Option 2',
      value: '2'
    }
  ];

  return (
    <Dropdown
      id='ads-dropdown'
      category='simple'
      selectorText='Select an option'
      selected={selected}
      options={options}
      onChange={setSelected}
      getItemLabel={item => item.label}
      getItemKey={item => item.value}
      getItemValue={item => item.value}
      getListTitle={item => item.label}
    />
  );
};

export default SimpleDropdown;
```
