## Getting Started

```javascript
import React, { Component } from 'react';
import { List } from '@codelittinc/agnostic-design-system';

class SimpleList extends Component {
  const list = [{
    label: 'Label',
    value: 'vlabel'
  },
  {
    label: 'Label 1',
    value: 'vlabel1'
  }]

  render() {
    return (
      <List
        options={list}
        id={'list'}
        getItemValue={(item) => item.value}
        getItemLabel={(item) => item.label}
        getItemKey={(item) => item.value}
        onChange={() => {}}
      />
    );
  }
}

export default SimpleList;

```

# Customazing styles

For the [List](https://design-system.codelitt.dev/?path=/story/components-list--simple-list) component we use the following [naming convention](https://design-system.codelitt.dev/?path=/story/welcome-customizing-styles--page) to configure the CSS variables:

**--list-[property]?-[attribute]-[state]?**

- Property - label, item, item-selected, item-label, item-cb, item-cm, item-icon, item-rb (optional)
- Attribute - background-color, border, font, etc (required)
- State - hover, focus, active (optional)

1. Create the css class using our variables

```css
.my-list-css {
  /* Component + Attribute */
  --list-border: 1px solid grey; /* Simplest variable */

  /* Component + Attribute */
  --list-height: auto;

  /* Component + Property + Attribute */
  --list-label-color: black;

  /* Component + Property + Attribute */
  --list-item-margin: blue;

  /* Component + Property + Attribute + State */
  --list-item-border-bottom-hover: 1px solid black;

  /* Component + Property + Attribute + State */
  --list-item-checkbox-background-color-checked: aquamarine;
}
```

2. Pass the css class through its `variablesClassName` property

`<List variablesClassName={styles['my-list-css']} />`
