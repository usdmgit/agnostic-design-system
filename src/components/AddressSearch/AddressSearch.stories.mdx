# Address Search Component

## Type: Decorator

The Address Search is based off the [Dropdown](https://design-system.codelitt.dev/?path=/docs/components-dropdown--default-dropdown) component.

## Getting Started

In order for the Address Search to function, it is necessary to wrap it with the MapsProvider component and provide
it with the google api key through the **apiKey** property. This is evident in the example below.<br />
The objective is to set the **_GOOGLE_API_KEY_** only once, in order to assist in the case the use of multiple Address
Search components is required.

Please refer to the following link for additional clarity: [Provider Concept](https://react-redux.js.org/api/provider)

```javascript
import React, { Component, useState } from 'react';
import { AddressSearch, MapsProvider } from '@codelittinc/agnostic-design-system';

class SimpleAddressSearch extends Component {
  const [value, setValue] = useState();

  render() {
    return (
      <MapsProvider apiKey={GOOGLE_API_KEY}>
        <AddressSearch
          selected={value}
          onChange={value => {
            setValue(value);
          }}
          placeholder='Enter Address'
          id='ads-address-search'
        />
      </MapsProvider>
    );
  }
}

export default SimpleAddressSearch;
```
