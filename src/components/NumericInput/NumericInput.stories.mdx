# Numeric Input Component

## Type: Decorator

The Numeric Input component is based off of the [Input](https://design-system.codelitt.dev/?path=/docs/components-input--default) component.

## Numeric Input Props

Given the nature of a decorator component, we use all the properties of the **Input**. Please refer to link above for more details.<br />
The Numeric Input, however, has four unique properties to it, as listed below:

|   Prop   |  Type   |                                                                                  Description                                                                                   | Default | Mandatory |
| :------: | :-----: | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------: | :-----: | :-------: |
|   max    | number  |                                                                    Maximum possible value for the component                                                                    |    -    |   false   |
| positive | boolean | Specifies whether or not the input can accept negative numbers. If props is absent, then it is allowed. If give a value of true, then negative numbers are unable to be typed. |    -    |   false   |
|  radix   | string  |                                             Fractional delimiter. For example: a radix of `','` with a `scale: 2` would be _1,00_                                              |    -    |   false   |
|  scale   | number  |                                                                  Digits after decimal point, `0` for integers                                                                  |    0    |   true    |

## Getting Started

```javascript
import React, { useState } from 'react';
import { NumericInput } from '@codelittinc/agnostic-design-system';

const SimpleNumericInput = () => {
  const [value, setValue] = useState('');

  return (
    <NumericInput
      label='Numeric Input'
      onChange={e => setValue(e.target.value)}
      max={15}
      positive={true}
      radix={','}
      scale={2}
      value={value}
    />
  );
};

export default SimpleNumericInput;
```
